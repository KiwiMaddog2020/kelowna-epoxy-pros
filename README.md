# Kelowna Epoxy Pros Website

Static preview site for a new Kelowna-based epoxy flooring business.

Live site: https://kiwimaddog2020.github.io/kelowna-epoxy-pros/

## Current Assumptions

- Brand name from supplied logo: `Kelowna Epoxy Pros`
- Market: Kelowna and the Central Okanagan
- Primary work: garage floors, shops, basements, patios, utility rooms, and light commercial spaces
- Contact details, legal business name, insurance, warranty, supplier names, testimonials, and real job photos are placeholders until Jeff confirms them

## Open Locally

Open `index.html` in a browser.

No build step is required. The site uses plain HTML, CSS, JavaScript, and local bitmap placeholder assets.

## Files

- `index.html`: single-page marketing site
- `styles.css`: visual system and responsive layout
- `script.js`: navigation, before/after slider, FAQ toggles, and preview form behavior
- `assets/`: generated bitmap placeholders, cropped swatches, and WebP delivery copies
- `assets/logo-dark.png`: cropped working logo for dark backgrounds
- `assets/logo-light.png`: cropped working logo for light backgrounds
- `assets/logo-transparent.png`: cleaned working logo for website placement
- `docs/brand-strategy.md`: brand, audience, voice, and copy rules
- `docs/content-ledger.md`: placeholders and facts still needed from Jeff
- `docs/launch-checklist.md`: practical checklist before going live

## Copy Rule

Consumer-facing copy should stay human, specific, and plain-spoken. Do not use em dashes. Do not add claims that Jeff has not confirmed.
