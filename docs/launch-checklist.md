# Launch Checklist

## Before Client Review

- Replace provisional contact details if Jeff has them
- Confirm `Kelowna Epoxy Pros` is the final public name
- Add at least one real finished floor photo if available
- Confirm which services should appear on the first screen

## Before Public Launch

- Confirm business name and domain
- Confirm phone, email, and form destination
- Confirm service area
- Confirm insurance and warranty wording
- Add privacy policy if the form stores or emails customer data
- Add real project photos with permission
- Add testimonials only with customer approval
- Check local SEO title and description
- Connect analytics only if Jeff wants it
- Test on mobile, tablet, and desktop

## Nice To Add Later

- Quote calculator for square footage ranges
- Care guide PDF
- Google Business Profile link
- Before and after project stories
- Supplier-backed coating system page
